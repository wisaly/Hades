Hades
Game of Life use Qt.

Tip:
Default map size is 128*128,random initialization.
Form title format:
Hades:envolution invertal:envolution count:cells alive
Keys:
r : restart game
c : kill all cells
u : make envolution speed up
d : make envolution speed down
o : make envolution speed to default(1000 milisecond per envalution)
spacebar : pause
g : toogle show grid lines
mouse left button : click to born a cell
mouse right button : click to kill a cell
